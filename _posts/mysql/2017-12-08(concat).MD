# <cente> concat 创建自定义显示字段

有时查询结果体现的格式不是表中的格式，若自定义查询格式，可使用concat。
> `concat (Value'(',Value,')')` AS 自定义字段名

`SELECT CONCAT (NAME,'(',wins,')') AS USER  FROM al_winner WHERE wins >=20`
